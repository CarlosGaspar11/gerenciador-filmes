## Iniciando o site

Inicialize com

```
npm run dev
```

Site desenvolvido para trabalho de faculdade.
A proposta é um site que você possa gerenciar filmes, adicionando e removendo de sua lista de favoristos, alem de marcar os que você ja assistiu.

Abra em: http://localhost:3000
Ou
Site: https://gerenciador-filmes-kappa.vercel.app
